## 📖 About the Project
This platform is a comprehensive educational web application designed to manage student profiles, track academic progress, and monitor school activities. A standout feature of this system is its innovative Persian reading education module. By leveraging AI technologies such as Text-to-Speech (TTS) and Speech-to-Text (Transcript), administrators can generate and prepare grade-specific interactive content to enhance students' reading skills. The entire system is backed by MongoDB to ensure flexible, scalable, and efficient database management.

## ✨ Key Features
- **Role-Based Architecture:** Dedicated, separate panels for "Students" and "Teachers," each with its own unique authentication and registration flow.
- **Real-Time Progress Tracking:** Enables both educators and learners to monitor grades, activities, and academic advancements instantly.
- **Comprehensive Teacher Dashboard:** Empowers educators with full authority to manage student profiles, oversee activities, and control the learning environment.
- **AI-Powered Learning:** Integrates TTS and Transcript services for an interactive Persian reading practice experience.

## 💻 Tech Stack
Built with a focus on high performance, type safety, and modern UI/UX principles:
- **Framework:** Next.js
- **Language:** TypeScript
- **Database:** MongoDB
- **Styling:** Tailwind CSS
- **State Management:** Redux
- **Deployment:** Vercel
